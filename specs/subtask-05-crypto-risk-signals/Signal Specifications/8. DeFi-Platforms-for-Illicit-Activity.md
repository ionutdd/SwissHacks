# Signal: DeFi Platforms for Illicit Activity  
**Priority:** Medium-High  
**Description:**  
Detect **DeFi protocols** used for money laundering (e.g., flash loans, synthetic assets).  
**Detection Methods:**  
- Monitor **unusual DeFi activity** (e.g., synthetic asset creation, arbitrage).  
- Flag **high-volume flash loans** (e.g., $1M+ in a single transaction).  
- Use **DeFi analytics tools** (e.g., Aave, Uniswap).  
**Real-World Example:**  
- **Ronin Bridge Hack (2022):** DeFi protocols were exploited to steal $625M.  
**Notes:**  
- Prioritize **DeFi security audits**.  
- Use **on-chain analytics** for synthetic asset tracking.
