# Signal: Wallets Linked to High-Risk Entities  
**Priority:** High  
**Description:**  
Identify wallets associated with **high-risk entities** (e.g., sanctioned organizations, dark web marketplaces).  
**Detection Methods:**  
- Cross-reference wallet addresses with **blacklist databases** (e.g., Chainalysis, Elliptic).  
- Monitor **dark web activity** (e.g., BTC transfers to Tor-based wallets).  
- Flag **unusual network activity** (e.g., transactions to hidden domains).  
**Real-World Example:**  
- **FTX Collapse (2022):** High-risk entities were linked to illicit fund transfers.  
**Notes:**  
- Use **real-time monitoring tools** for wallet activity.  
- Integrate with **regulatory compliance APIs**.
