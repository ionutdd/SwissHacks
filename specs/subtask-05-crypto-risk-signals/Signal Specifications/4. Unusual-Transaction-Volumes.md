# Signal: Unusual Transaction Volumes  
**Priority:** Medium-High  
**Description:**  
Flag transactions with **abnormal volume** or **pattern** (e.g., sudden large transfers, repeated small deposits).  
**Detection Methods:**  
- Set **thresholds** for transaction frequency and size.  
- Use **machine learning** to detect anomalies (e.g., sudden spikes in activity).  
- Monitor **whale activity** (e.g., large wallets moving funds).  
**Real-World Example:**  
- **Terra/Luna Collapse (2022):** Unusual volume spikes in UST led to a $40B collapse.  
**Notes:**  
- Combine with **social media monitoring** for real-time alerts.  
- Use **transaction graph analysis** to trace patterns.
