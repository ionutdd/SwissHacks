# Signal: Cross-Chain Bridges  
**Priority:** Medium-High  
**Description:**  
Detect transactions that move funds across **blockchains** via **bridges** (e.g., Wormhole, Chainlink).  
**Detection Methods:**  
- Monitor **bridge-specific addresses** (e.g., Wormhole’s smart contracts).  
- Analyze **cross-chain transaction patterns** (e.g., deposit on Ethereum, withdrawal on Solana).  
- Flag **sudden volume spikes** in bridge activity.  
**Real-World Example:**  
- **Wormhole Exploit (2022):** Hackers exploited a bridge to steal $600M in stablecoins.  
**Notes:**  
- Use **multi-chain analytics tools** (e.g., Blockchair, Blockchair).  
- Prioritize **security audits** for bridge protocols.
